# EZVals Skill

Agent skill for writing, running, and analyzing LLM evaluations using [EZVals](https://github.com/camronh/EZVals).

## Install

```bash
npx skills add camronh/ezvals-skill
```

Or install from the EZVals package (version-matched):

```bash
pip install ezvals
ezvals skills add
```

## Usage

Invoke with `/ezvals` in your AI coding agent (Claude Code, Cursor, Codex, etc.).
